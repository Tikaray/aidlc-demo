# AIDLC演示产品 — 需求文档

## Intent Analysis

- **用户请求**: 制作一个演示产品向领导展示AIDLC开发方式的优势和在工作中的应用
- **请求类型**: New Project（全新项目）
- **范围**: 展示型Web应用（PPT风格幻灯片）
- **复杂度**: Low-Moderate（展示型应用，内容驱动）

---

## 1. 项目概述

构建一个基于React的PPT风格网页演示产品，面向技术领导展示AIDLC（AI驱动开发生命周期）的优势和在实际工作中的应用。演示以案例为核心，涵盖产品设计与开发、新媒体运营两大工作场景，结合个人心得体会，通过图文、数据、实操演示等综合方式呈现。

## 2. 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端框架 | React |
| 幻灯片引擎 | 自定义React幻灯片组件（轻量实现） |
| 样式 | CSS Modules / 内联样式 |
| 构建工具 | Vite |
| 部署 | 本地开发环境（仅桌面端） |

## 3. 目标受众

| 受众 | 关注点 |
|------|--------|
| 技术领导 | 技术实现细节、开发效率提升、AI工具在工程实践中的价值 |

## 4. 功能需求

### 4.1 幻灯片导航系统

- FR-001: 支持键盘左右箭头切换幻灯片
- FR-002: 支持点击导航按钮前进/后退
- FR-003: 显示当前页码/总页数
- FR-004: 支持幻灯片概览/目录页，可点击跳转

### 4.2 内容展示模块

- FR-005: 封面页（标题、副标题、演讲者信息）
- FR-006: 目录页（演示大纲，可点击跳转）
- FR-007: AIDLC方法论介绍页（流程图/示意图展示AIDLC各阶段）
- FR-008: 案例展示页 — 产品设计与开发场景
  - AI辅助需求分析案例
  - AI辅助架构设计案例
  - AI辅助代码生成案例
- FR-009: 案例展示页 — 新媒体运营场景
  - AI辅助内容策划案例
  - AI辅助活动方案案例
  - AI辅助文案生成案例
- FR-010: 心得体会页（Before/After对比 + 效率数据 + 实操总结）
- FR-011: 总结与展望页

### 4.3 交互式代码演示

- FR-012: 嵌入式代码展示区域（语法高亮）
- FR-013: 代码执行结果/效果预览区域
- FR-014: Before/After代码对比展示

### 4.4 数据可视化

- FR-015: 效率提升数据图表（如柱状图/饼图展示时间节省）
- FR-016: 工作流程对比图（传统方式 vs AIDLC方式）

### 4.5 视觉效果

- FR-017: 幻灯片切换动画（淡入淡出/滑动）
- FR-018: 内容元素进入动画（逐步显示）
- FR-019: 统一的视觉主题（专业商务风格）

## 5. 非功能需求

- NFR-001: 界面语言为中文
- NFR-002: 仅支持桌面端（1920x1080投屏优化）
- NFR-003: 页面切换流畅，无卡顿
- NFR-004: 无需后端服务，纯前端应用
- NFR-005: 1-3天内完成开发

## 7. MVP范围边界

### 包含
- React幻灯片导航系统
- 15-20页演示内容
- 交互式代码展示
- 数据可视化图表
- 幻灯片切换动画
- 键盘导航

### 不包含
- 后端服务
- 用户登录/权限
- 内容编辑功能
- 导出PDF功能
- 演讲者备注视图
- 多主题切换
